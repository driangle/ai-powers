# Instructions

Keep `README.md` in sync with the skills directory. Update it whenever a skill is added, removed, or changed in a way that affects its description.

When adding a new skill, also register it in `.claude-plugin/marketplace.json` under the appropriate plugin. If no existing plugin fits, create a new one.

Bump the `version` in `.claude-plugin/marketplace.json` whenever skills are added, removed, or changed.
